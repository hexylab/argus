# Argus - プロジェクト概要

## 大目標

**映像AIモデルの学習データ生成から学習までを全自動で行うSaaSシステム**

ユーザーが自然言語で「〇〇を検出するモデルを作成してください」と依頼するだけで、AIエージェントが映像データの探索・アノテーション・学習・評価を自動実行し、Human-in-the-Loopで品質を担保しながら高精度なモデルを提供する。

## ビジョン

```
従来のMLワークフロー:
  データ収集 → 手動アノテーション → 学習 → 評価 → 改善 (数週間〜数ヶ月)
                    ↑ ここがボトルネック

Argusのワークフロー:
  「このラベルでモデル作って」→ AIが自動処理 → 確認依頼 → 完成 (数時間〜数日)
```

## コアバリュー

1. **自然言語インターフェース**: 専門知識不要でモデル作成を依頼可能
2. **Human-in-the-Loop**: AIが判断に迷う箇所のみユーザーに確認、効率と品質を両立
3. **End-to-End自動化**: データ探索から学習済みモデル配信まで一貫して自動化
4. **段階的スケール**: 小規模から始めて大規模まで同じアーキテクチャで対応

## 対象ユーザー

- 製造業の品質管理担当者
- 研究機関のデータサイエンティスト
- MLエンジニアリングリソースが限られたスタートアップ
- カスタム物体検出が必要な企業

## 対応タスク（段階的拡張）

| フェーズ | タスク |
|---------|--------|
| 初期 | 物体検出（バウンディングボックス） |
| 中期 | セグメンテーション |
| 将来 | 動作認識、姿勢推定 |

## 成功指標

- アノテーション工数: 従来比 80% 削減
- モデル作成リードタイム: 数週間 → 数日
- ユーザー満足度: 確認依頼の適切さ、最終モデル精度
